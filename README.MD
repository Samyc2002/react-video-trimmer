# react-video-trimmer

Amazing React component for manipulate videos lenght

## Installation

```bash
npm install --save react-video-trimmer
```

or:

```bash
yarn add react-video-trimmer
```

## Usage

```jsx
import ReactVideoTrimmer from "react-video-trimmer";
<div>
  <ReactVideoTrimmer />
</div>;
```
